# SAP S/4HANA ABAP Cloud Mini Project

## Description
This academic mini project demonstrates the architecture of a modern SAP S/4HANA
application using ABAP CDS Views, OData services, and SAP Fiori Elements.

Due to SAP license restrictions, the project was developed in a simulated
ABAP Cloud environment while following SAP best practices.

## Architecture
CDS View → OData Service → SAP Fiori Elements (List Report)

## Technologies
- ABAP CDS
- OData
- SAP Fiori Elements
- Eclipse ADT

## Note
This project is intended for learning and demonstration purposes.
